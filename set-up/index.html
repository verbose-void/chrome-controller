Set Up Instructions

Getting set up with Chrome Controller can be done in 3 easy steps.

  1. Acquire a controller that is compatible with Gamepad API. The good news is it supports over 1,000 controllers. Chrome Controller has the best support available for Xbox/PlayStation controllers, however you can still customize button mappings for any 3rd party controller. View an exhaustive list of supported controllers here.

  2. Before adding Chrome Controller to your browser, test that your controller is detected by the Gamepad API here with no download/installation required.

  3. Add Chrome Controller to your browser, sit back and relax!
